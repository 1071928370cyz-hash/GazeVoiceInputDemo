# 公司电脑运行步骤（傻瓜清单）

> 给代码小白看的。照着从上往下做即可。出问题看最后「卡住了怎么办」。

---

## 0. 这个 App 是干嘛的

单手握手机时，顶部搜索框够不着、不好打字。本 Demo：
**你看屏幕 → 顶部输入框展开、握持手那侧浮出语音按钮 → 点一下说话、再点一下结束 → 出搜索结果。**

三个能力：① 看没看屏幕（Swing 注视）② 哪只手握着（智感握姿）③ 语音转文字。

---

## 1. 开始前确认（很重要）

- [ ] 这台电脑装了 **DevEco Studio**。
- [ ] 这台电脑装了 **Swing（swingability）SDK**（公司内部环境才有）。
      ⚠️ 没有它，整个工程**编译不过**——这是注视功能的硬性要求，不是 bug。
- [ ] 有一台**带 Swing 能力的华为真机**（注视/握姿是手机硬件能力，**模拟器测不了**）。
- [ ] 手机用数据线连到电脑，手机里**开发者模式 + USB 调试**已打开。

---

## 2. 打开工程

1. 打开 **DevEco Studio**。
2. 顶部菜单 `File → Open`。
3. 选中 `GazeVoiceInputDemo` 这个**文件夹**，点开。
4. 右下角会自动转圈下载依赖、建临时文件，**等它转完**（第一次可能要几分钟）。

---

## 3. 配置签名（让 App 能装进手机）

1. 顶部菜单 `File → Project Structure`。
2. 左边选 `Signing Configs`。
3. 勾上 **`Automatically generate signature`（自动生成签名）**，按提示登录华为开发者账号。
4. 等它显示签名成功，点 `OK`。

> 注视用到的权限 `SUBSCRIBE_SWING_ABILITY` 可能需要**更高级别的系统签名**。
> 如果第 6 步注视一直「订阅失败」，就是这里——找公司里**管签名证书的同事**帮忙配，普通调试签名可能不够。

---

## 4. 选中手机并运行

1. 看 DevEco **右上角** ▶（运行）按钮左边的下拉框，确认：
   - 模块选的是 `entry`；
   - 设备选的是**你的华为手机**（不是模拟器）。
2. 点 **▶ 运行**。
3. 第一次装会让你在**手机上**点「允许安装」。
4. App 启动后，会弹权限请求：**麦克风、手势识别**——全部点**允许**。

---

## 5. 怎么操作这个 App

1. **看着屏幕**（或点一下顶部搜索框）→ 顶部展开输入框，握持手那侧浮出蓝色语音按钮。
2. **点一下语音按钮** → 变红、有声浪动效，开始听你说话，文字实时出现在输入框。
3. 说完 **再点一下** → 结束，下方弹出「视频/图片/相关」搜索结果。

---

## 6. 怎么判断「注视」接成功了（看左上角黑色状态条）

| 状态条显示 | 意思 |
| --- | --- |
| `注视:Swing` | ✅ 成了，真眼动在控制 |
| `注视:手动` | ❌ 没订阅上，正在用「点搜索框=注视」兜底 |
| `gaze: Swing 订阅成功` | ✅ 系统接受了订阅 |
| `gaze: Swing 失败(数字)…` | ❌ 被拒，记下那个**数字**（错误码） |
| `原始: {…eyeGaze…}` 绿字会跳动 | ✅ 硬件在上报；盯屏幕/转头看它变化 |

**首次务必核对**：盯着屏幕、再把头转开，看绿色「原始:」那行里**有没有 `eyeGaze` 这个词、数值是不是在 1/0/2 之间跳**（1=在看，0=没看，2=没人脸）。对得上就说明接对了。

---

## 7. 卡住了怎么办

| 现象 | 处理 |
| --- | --- |
| 编译报错，提示找不到 `@ohos.swingability` | 这台电脑没装 Swing SDK，换有 SDK 的公司环境 |
| ▶ 是灰的点不动 | 右下角还在转圈（等它）/ 没选手机设备 |
| 改了代码看不到变化 | `Build → Clean Project`，再运行；必要时手机上卸载旧 App 重装 |
| `注视:手动` 一直变不成 `Swing` | 多半是**签名等级不够**（看第 3 步提醒），找管签名的同事 |
| 不知道哪错了 | 把状态条那几行（尤其 `gaze:` 的失败码 + `原始:` 的绿字）拍照发出来求助 |

## 8. 文件在哪（万一要找人改代码）

```
entry/src/main/ets/
├── pages/Index.ets             # 主页面（把三个能力串起来）
├── components/VoiceButton.ets  # 语音按钮（点一下切换 + 声浪）
└── services/
    ├── HandDetector.ets        # 握持手检测
    ├── SpeechService.ets       # 语音转文字
    ├── GazeDetector.ets        # 注视（调用下面的 Swing）
    └── gaze/
        ├── swing.ts            # Swing 底层（勿改）
        └── SwingGaze.ets       # Swing 封装
entry/src/main/module.json5     # 权限声明
```
