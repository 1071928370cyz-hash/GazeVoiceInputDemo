# 注视后端「一键开关」说明

注视有两种后端，默认是**【关】**（本机就能编译安装）。切换只动 **2 个地方**。

---

## 现在的默认状态：【关】= 手动兜底（本机可用）

- 不依赖 Swing SDK、不需要系统权限，**你自己的电脑就能编译、装进手机**。
- 注视用「点一下顶部搜索框 = 注视」来代替，握姿 + 语音功能完全正常。
- 适合：在本机先把 App 跑起来、演示握姿和语音。

> 之前的「安装失败 failed to install bundle」就是因为开着 Swing 权限、签名授不下来。
> 现在默认关掉了，应该能正常安装。

---

## 到公司电脑（有 Swing SDK）启用真注视：【开】

做下面 **两步**，然后重新运行：

### 第 1 步：`entry/src/main/ets/services/gaze/swing.ts`
- 把文件里 **【关】整段** 用 `/*  */` 注释掉；
- 把 **【开】整段** 最外层的 `/*` 和 `*/` 删掉（取消注释）。
- 文件顶部有详细图示，照着做。

### 第 2 步：`entry/src/main/module.json5`
- 找到带「注视开关·第 2 步」注释的位置；
- 取消 `SUBSCRIBE_SWING_ABILITY` 那几行的注释。

### 然后
- `Build → Clean Project` → 连真机 → 运行。
- 看左上角状态条：变成 `注视:Swing`、`gaze: Swing 订阅成功` 就成了。
- 若还是 `注视:手动` 或安装失败，多半是**签名等级不够**，找公司管签名证书的同事配 ACL / 系统签名。

---

## 想从【开】切回【关】

把上面两步反过来即可（swing.ts 注释回【关】、module.json5 注释回权限）。
或者直接用 git 还原这两个文件。
